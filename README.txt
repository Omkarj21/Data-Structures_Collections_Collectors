# Python Data Structures/Collections/Collectors :->

# List is a collection which is ordered and changeable, Allows duplicate members, denoted by [], separated by commas, use the slicing operator [ ] to extract an item or a range of items.
# Tuple is a collection which is ordered and unchangeable, Allows duplicate members, denoted by (), separated by commas, use the slicing operator [ ] to extract an item or a range of items.
# Set is a collection which is unordered, changeable and unindexed. Only Unique members, No duplicate members. denoted by {}, separated by commas, slicing operator [] does not work as it is unindexed.
# Dictionary is a collection which is unordered, changeable. No duplicate members. {Key:Value}, separated by commas.
# Strings are immutable, use single quotes or double quotes to represent strings. Multi-line strings can be denoted using triple quotes, ''' or """.
